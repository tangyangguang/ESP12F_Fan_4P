## 3. 电源设计

| 参数 | 数值 | 说明 |
|------|------|------|
| 输入电压 | 12V DC | 外置电源适配器，≥5A 输出能力 |
| 系统工作电压 | 5V DC | MP2307 DC-DC 输出，效率≥92%，最大输出 3A |
| MCU 工作电压 | 3.3V DC | AMS1117-3.3 LDO 稳压，最大输出 800mA |
| 风扇供电 | 12V DC / 5A | 直接从输入电源取电，与 MCU 供电回路独立 |
| 最大总电流 | ≤4A | 风扇峰值电流 3A + 系统工作电流 0.3A |

## 4. 关键电路说明

### 4.1 风扇驱动电路

- **MOSFET**：IRLR7843 N 沟道 MOS 管，Vgs 阈值 1.8V，可直接 3.3V GPIO 驱动
- **导通电阻**：1.7mΩ，3A 工作电流下功耗仅 15mW，无需额外散热
- **栅极电阻**：100Ω 串联，抑制开关振荡，防止误导通
- **续流二极管**：SS54 肖特基二极管，吸收风扇关断时反向电动势，靠近 MOS 管放置
- **PWM 频率**：25KHz，避开人耳听觉范围，无啸叫噪声

### 4.2 红外接收电路

- **接收头**：VS1838 一体化接收头，内置放大、滤波、38KHz 解调
- **抗干扰**：电源引脚串联 100Ω 电阻 + 100nF 电容滤波
- **输出**：直接输出数字信号到 MCU GPIO13
- **协议支持**：NEC、SONY、RC5、RC6、SAMSUNG 等常见协议，通过 IRremoteESP8266 库解码

### 4.3 按键电路

- **类型**：微动按键 ×2（加速键 GPIO14、减速键 GPIO4）
- **上拉**：ESP8266 内部上拉，无需外部电阻
- **防抖**：软件防抖，检测按下后延时 50ms 确认
- **BOOT 键**：GPIO0，长按 1s 清除 WiFi 凭证，上电时内部上拉确保正常启动

### 4.4 LED 指示灯电路

- **位置**：ESP-12E 模块内置板载 LED，连接到 GPIO2
- **电平逻辑**：低电平亮（GPIO2 输出 LOW 时亮）
- **PWM 调光**：支持 `analogWrite()` PWM 调光，0 档熄灭，1-4 档对应 25%/50%/75%/100% 亮度
- **闪烁模式**：慢闪（1Hz）表示 WiFi 未连接，快闪（5Hz）表示故障，单次闪表示操作反馈

## 6. PCB 设计检查清单

> **打样前逐条检查，[ ] → [x]**

### 布局
- [ ] 数字/强电区域分区隔离，风扇驱动电路远离 MCU
- [ ] 去耦电容距 IC 电源引脚≤2mm
- [ ] ESP-12F 天线区域无铜箔无遮挡，周围留出 3mm 以上净空
- [ ] 大电流路径（风扇回路）布局在板边
- [ ] 按键布局在易于操作的位置

### 布线
- [ ] 双层板有完整地平面，不随意分割
- [ ] 12V 电源线宽≥5mm，5V≥2mm，3.3V≥1mm（1 盎司铜厚）
- [ ] 关键信号（PWM 输出、转速反馈）预留测试点
- [ ] 电源地和信号地单点连接，避免地环路

### EMI 与防护
- [ ] 风扇 PWM 信号不与通信/输入信号平行走线
- [ ] 各电源入口 100nF + 大容量电解去耦组合
- [ ] 所有对外接口（电源、风扇）有 ESD/浪涌保护
- [ ] 风扇感性负载并联续流二极管，靠近 MOS 管放置
- [ ] 3.3V 与 12V 之间满足爬电距离≥2mm
