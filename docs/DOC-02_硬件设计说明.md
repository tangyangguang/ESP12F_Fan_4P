## 4. 关键电路说明

### 4.1 四线风扇控制电路

- **供电方式**：四线风扇 12V 与 GND 常供电，不通过低边 MOS 管切断风扇电源。
- **PWM 控制**：风扇 PWM 控制脚按 Intel 4-wire PWM 习惯使用开漏/开集电极下拉方式，频率 25KHz；软件固定 `analogWriteRange(255)`，硬件驱动级必须保证 Web 语义 0%=停止、100%=全速，不得在风扇端形成反相占空比。
- **电平保护**：ESP8266 GPIO 不直接承受风扇内部 5V 上拉；建议使用小信号 NPN/NMOS 做开漏下拉，栅极/基极串联限流电阻。
- **TACH 反馈**：TACH 通常为开集电极输出，应上拉到 3.3V 后接入 GPIO12，避免 5V 进入 ESP8266。
- **保护器件**：12V 风扇供电入口保留 TVS、保险丝和去耦；PWM/TACH 信号预留串联电阻和必要的 ESD 防护。

### 4.2 红外接收电路

- **接收头**：VS1838 一体化接收头，内置放大、滤波、38KHz 解调
- **抗干扰**：电源引脚串联 100Ω 电阻 + 100nF 电容滤波
- **输出**：直接输出数字信号到 MCU GPIO13
- **协议支持**：NEC、SONY、RC5、RC6、SAMSUNG 等常见协议，通过 IRremoteESP8266 库解码

### 4.3 按键电路

- **类型**：微动按键 ×2（加速键 GPIO14、减速键 GPIO4）
- **上拉**：ESP8266 内部上拉，无需外部电阻
- **防抖**：软件防抖，检测按下后延时 50ms 确认
- **BOOT 键**：GPIO0，长按 1s 清除 WiFi 凭证，上电时内部上拉确保正常启动

### 4.4 LED 指示灯电路

- **位置**：ESP-12E 模块内置板载 LED，连接到 GPIO2
- **电平逻辑**：低电平亮（GPIO2 输出 LOW 时亮）
- **PWM 调光**：支持 `analogWrite()` PWM 调光，0 档熄灭，1-4 档对应 25%/50%/75%/100% 亮度
- **闪烁模式**：慢闪（1Hz）表示 WiFi 未连接，快闪（5Hz）表示故障，单次反相脉冲表示操作反馈
- **状态优先级**：故障快闪 > WiFi 未连接慢闪 > 档位亮度/熄灭；故障状态下操作反馈不会清除快闪
- **反馈范围**：Web、按键、红外和配置保存且有变化等有效操作闪 1 下；默认 200ms，可配置 0-2000ms，0 表示关闭操作反馈；当前亮时短暂熄灭，当前灭时短暂点亮；非法参数、配置无变化、边界重复加减档、未知红外码和学习超时不闪

## 6. PCB 设计检查清单

> **打样前逐条检查，[ ] → [x]**

### 布局
- [ ] 数字/强电区域分区隔离，风扇驱动电路远离 MCU
- [ ] 去耦电容距 IC 电源引脚≤2mm
- [ ] ESP-12F 天线区域无铜箔无遮挡，周围留出 3mm 以上净空
- [ ] 大电流路径（风扇回路）布局在板边
- [ ] 按键布局在易于操作的位置

### 布线
- [ ] 双层板有完整地平面，不随意分割
- [ ] 12V 电源线宽≥5mm，5V≥2mm，3.3V≥1mm（1 盎司铜厚）
- [ ] 关键信号（PWM 输出、转速反馈）预留测试点
- [ ] 电源地和信号地单点连接，避免地环路

### EMI 与防护
- [ ] 风扇 PWM/TACH 信号不与通信/输入信号长距离平行走线
- [ ] 各电源入口 100nF + 大容量电解去耦组合
- [ ] 所有对外接口（电源、风扇）有 ESD/浪涌保护
- [ ] 四线风扇 12V 常供电，PWM/TACH 均通过 3.3V 安全电平连接 MCU
- [ ] 3.3V 与 12V 之间满足爬电距离≥2mm
